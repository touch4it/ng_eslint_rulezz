### NG esling rulezz

ESlint rules for angular applications.

## Compatibility

| Lib version | Angular version |
|:------------|:----------------|
| ~1.0.0      | >=13.0.0        |
| ~1.1.0      | 15.0.0          |
| ~1.2.0      | 16.0.0          |
| ~1.3.0      | 17.0.0          |
| ~1.4.0      | 18.0.0          |

## Usage
Add this to your `.eslintrc.json`
```json
{
  "extends": "./node_modules/@touch4it/ng_eslint_rulezz"
}
```
